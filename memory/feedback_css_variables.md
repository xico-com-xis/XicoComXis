---
name: CSS variable naming preference
description: User prefers semantic/plain color names over palette scale names (like slate-900)
type: feedback
---

Use plain descriptive names for CSS variables (e.g. --color-primary, --color-text) instead of palette scale names like --slate-900 or --slate-800.

**Why:** User finds scale-based names (slate, etc.) unintuitive.
**How to apply:** Always name color variables semantically when writing or suggesting CSS custom properties.
