# AGENTS

## Scope
This file applies to the entire repository.

## Project overview
This is a static HTML/CSS/JavaScript project with a single-page layout.

## Pull Request Guidelines 
- Create a new branch for every task. 
- Include a summary of changes in the PR description.

## Development guidelines
- Prefer plain HTML, CSS, and vanilla JavaScript (no frameworks or build tools).
- Keep changes minimal and consistent with existing typography, spacing, and color usage.
- Match existing formatting (two-space indentation in HTML/CSS; semicolons in JS).
- If you add assets, place them in the `images/` directory and reference them with relative paths.

## Testing
- No automated tests are configured. Manually open `index.html` in a browser to validate changes.
