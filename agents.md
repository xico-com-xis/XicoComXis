## Best Practices

When working with agents in this repository:

1. **Clear Instructions**: Provide specific, actionable tasks for agents
2. **Scope Definition**: Define the scope of changes clearly
3. **Review Changes**: Always review AI-generated changes before merging
4. **Incremental Updates**: Make small, focused changes rather than large refactors
5. **Documentation**: Keep this file updated as new agents or automation are added
